Rules + AI Enrichment

Eligibility gates · Gemini 2.5 Flash
Hard rule · Non-negotiable
≤ 15s
Maximum duration
Every clip must have a known duration. Unknown = ineligible. No exceptions across any platform.
Niche sports threshold
Kabaddi · Chess · Field Hockey · Athletics
8–20s window · 7-day lookback · 1K min views (vs 10–15s · 3 days · 5K mainstream).
Deduplication
Exact + repost detection
sha256 URL hash applied across all platforms. Canonical keys: reddit:<id> · x:<id> · twitch:<id>.
AI model
Gemini 2.5 Flash on Vertex AI
GA stable. Project ai-automation-491011 · region asia-southeast1. Lazy-init, reuses service account.
Output 1
Caption — 120 chars max
Punchy, sport-specific, no emojis. Generated after dedup pass. Fallback to empty string — never blocks pipeline.
Output 2
8 hashtags · JSON
Exactly 8 relevant hashtags per clip. Returns structured raw JSON. India team may edit before approving.
